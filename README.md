# Tripleten web_project_around_es

Around The U.S. es una aplicación web interactiva que permite al usuario editar su perfil y compartir lugares mediante tarjetas con imágenes. El proyecto está enfocado en la manipulación del DOM usando JavaScript, el manejo de ventanas emergentes y la generación dinámica de contenido a partir de datos.

# Funcionalidad

- Edición del perfil del usuario mediante un formulario en un modal.
- Renderizado dinámico de tarjetas a partir de un arreglo de datos inicial.
- Agregar nuevas tarjetas usando un formulario.
- Eliminar tarjetas del DOM.
- Dar "Me gusta" a las tarjetas.
- Visualizar una imagen ampliada al hacer clic sobre ella.
- Abrir y cerrar modales reutilizando funciones comunes.

# Tecnologías

- HTML5
- CSS3
- JavaScript
